# SuperMind 平台策略库完整索引

## 📊 平台概览

基于对SuperMind量化交易平台的深入调研，本文档提供了平台所有策略模块的完整索引和使用指南。

## 🗂️ 策略模块结构

### 一、我的策略 (My Strategies)

#### 1. 策略研究 (Strategy Research)
- **功能**: 基于Jupyter Notebook的策略开发环境
- **特点**:
  - 支持Python 3.5/3.8环境
  - 实时代码调试和回测
  - 丰富的数据科学库支持
  - 可视化图表生成
- **访问**: https://quant.10jqka.com.cn/view/study-index.html

#### 2. 指标策略 (Indicator Strategies)
- **功能**: 技术指标策略开发模块
- **支持指标类型**:
  - 股票指标策略
  - 期货指标策略
- **特点**:
  - 内置技术指标计算函数
  - 支持自定义指标
  - 参数优化功能
- **状态**: 需要登录查看具体策略列表

#### 3. 因子策略 (Factor Strategies)
- **功能**: 量化因子策略开发
- **访问**: https://quant.10jqka.com.cn/view/machine.html#/newStudyMf
- **特点**:
  - 因子挖掘和分析
  - 多因子模型构建
  - 因子有效性检验

#### 4. 向导式策略 (Wizard Strategies)
- **功能**: 图形化策略构建工具
- **访问**: https://quant.10jqka.com.cn/view/machine.html#/guideGeneratorStrategyList
- **特点**:
  - 无需编程基础
  - 拖拽式策略构建
  - 模板化策略选择

#### 5. 策略监控 (Strategy Monitoring)
- **功能**: 实时策略运行监控
- **访问**: https://quant.10jqka.com.cn/view/strategymonitor.html
- **特点**:
  - 实时绩效跟踪
  - 风险指标监控
  - 异常情况预警

#### 6. 策略库 (Strategy Library)
- **功能**: 策略存储和管理中心
- **访问**: https://quant.10jqka.com.cn/view/strategylib.html
- **特点**:
  - 策略版本管理
  - 策略分类存储
  - 社区策略分享

### 二、因子研究模块

#### 1. 因子检测 (Factor Detection)
- **访问**: https://quant.10jqka.com.cn/view/machine.html#/factorDetectionList
- **功能**: 因子有效性检测和分析

#### 2. 因子库 (Factor Library)
- **访问**: https://quant.10jqka.com.cn/view/machine.html#/factorLibrary
- **功能**: 因子数据库查询和使用

#### 3. 绩效分析 (Performance Analysis)
- **访问**: https://quant.10jqka.com.cn/view/machine.html#/performanceAnalysis
- **功能**: 策略绩效深度分析

### 三、实盘交易模块

#### 1. 模拟交易 (Simulation Trading)
- **功能**: 模拟真实交易环境
- **特点**:
  - 支持日级策略
  - 支持分钟级策略
  - 实时订单撮合机制
  - Webhook消息推送
- **创建方式**: 客户端 → 量化交易 → 模拟交易 → 新建模拟交易

#### 2. 指标策略模拟 (Indicator Strategy Simulation)
- **功能**: 技术指标策略模拟交易
- **特点**: 专门针对技术指标策略的模拟环境

#### 3. Python策略模拟 (Python Strategy Simulation)
- **访问**: https://quant.10jqka.com.cn/view/trade.html
- **功能**: Python策略模拟交易环境

## 📈 策略开发流程

### 第一步：策略设计
1. 选择策略类型（趋势/震荡/套利）
2. 确定技术指标组合
3. 设定参数范围
4. 制定风控规则

### 第二步：策略开发
1. 在**策略研究**模块编写代码
2. 使用内置API获取数据
3. 实现技术指标计算
4. 编写交易逻辑

### 第三步：回测验证
1. 设置回测参数（时间段、初始资金等）
2. 运行历史数据回测
3. 分析绩效指标
4. 参数优化调整

### 第四步：模拟交易
1. 创建**模拟交易**任务
2. 实时监控策略表现
3. 调整优化参数
4. 验证实盘可行性

### 第五步：实盘部署
1. 申请实盘交易权限
2. 部署策略到实盘环境
3. 实时监控和风险控制
4. 持续优化调整

## 🔧 核心功能详解

### 策略研究环境

**技术栈**:
- Jupyter Notebook (Python 3.5/3.8)
- Pandas, NumPy, Matplotlib
- SuperMind专属API

**核心API**:
```python
# 数据获取
get_price() - 获取价格数据
get_fundamental() - 获取基本面数据
query_iwencai() - 问财自然语言查询

# 技术指标
talib库 - 150+技术指标
自定义指标计算

# 回测函数
research_strategy() - 策略回测
research_trade() - 实盘交易
```

### 模拟仿真系统

**运行机制**:
- **日级策略**: 每日收盘后执行
- **分钟策略**: 每分钟执行一次
- **Tick策略**: 实时行情触发执行

**撮合机制**:
- 基于真实历史数据
- 考虑交易成本和滑点
- 支持订单队列模拟

**监控功能**:
- 实时收益曲线
- 交易明细记录
- 持仓情况跟踪
- 运行日志查看

### 向导式策略构建

**适用人群**:
- 无编程基础的量化初学者
- 需要快速验证想法的专业人士
- 教育和研究用途

**构建流程**:
1. 选择策略模板
2. 设置参数
3. 选择标的范围
4. 配置风控规则
5. 一键生成代码
