## 🎯 技术指标策略页面访问指南

### 方法一：通过主菜单导航
1. 登录SuperMind平台
2. 点击顶部导航"我的策略"
3. 选择"指标策略"选项

### 方法二：直接访问（需要登录）
```
主页面 → 我的策略 → 指标策略
https://quant.10jqka.com.cn/view/
```

### 方法三：通过策略库访问
1. 进入策略库页面
2. 在策略目录中选择技术指标相关策略
3. 点击具体策略查看详情

## 📊 策略模块详细说明

### 1. 指标策略模块
**访问路径**: 我的策略 → 指标策略

**主要功能**:
- 内置技术指标计算
- 参数优化工具
- 多指标组合策略
- 回测和模拟交易

**支持的技术指标**:
- **趋势类**: MA, EMA, MACD, BOLL
- **震荡类**: RSI, KDJ, CCI, WR
- **成交量类**: OBV, VOL, 量比
- **volatility类**: ATR, 布林带宽度

### 2. 向导式策略模块
**访问路径**: 我的策略 → 向导式策略

**特点**:
- 无需编程基础
- 拖拽式界面
- 预设策略模板
- 一键生成代码

**策略模板类型**:
- 趋势跟踪策略
- 均值回归策略
- 突破策略
- 套利策略

### 3. 因子策略模块
**访问路径**: 我的策略 → 因子策略

**功能**:
- 因子挖掘和分析
- 多因子模型构建
- 因子有效性检验
- 因子组合优化

## 🆘 常见访问问题

### Q1: 页面显示"暂无内容"
**解决方案**:
- 确认已登录账号
- 检查账号权限等级
- 尝试刷新页面
- 联系客服确认状态

### Q2: 无法访问策略库
**解决方案**:
- 完成实名认证
- 提升账号权限等级
- 确认网络连接正常
- 清除浏览器缓存

### Q3: 模拟交易无法创建
**解决方案**:
- 先创建策略并完成回测
- 确保策略代码无错误
- 检查账号权限
- 联系技术支持

### Q4: API调用失败
**解决方案**:
- 检查API权限
- 确认参数格式正确
- 查看错误日志
- 参考官方文档
