# 股票1分钟涨速监控程序

## 功能介绍
这个程序可以实时监控股票市场，发现1分钟涨速前100的股票，帮助快速识别市场热点。

## 主要功能
- 实时获取股票行情数据
- 计算1分钟涨速
- 筛选涨速前100的股票
- 实时显示排行榜
- 自动保存历史数据

## 使用方法

### 1. 安装依赖
```bash
pip install requests pandas
```

### 2. 配置API
编辑 `config.json` 文件，设置你的同花顺API token：
```json
{
    "api_config": {
        "token": "你的API_TOKEN"
    }
}
```

### 3. 运行程序
```bash
python stock_monitor.py
```

### 4. 参数说明
- `interval`: 监控间隔（秒），默认60秒
- `top_n`: 显示前N个股票，默认100
- `save_results`: 是否保存结果到文件

## 输出格式
程序会在控制台实时显示涨速排行榜，包含以下信息：
- 排名
- 股票代码
- 股票名称
- 现价
- 涨跌幅
- 1分钟涨速
- 成交量

## 数据文件
程序会自动保存数据到JSON文件，文件名格式：`top_rising_stocks_YYYYMMDD_HHMMSS.json`

## 注意事项
1. 需要有效的同花顺API token才能获取真实数据
2. 没有token时程序会使用模拟数据进行演示
3. 建议在交易时间内运行以获取最佳效果
4. 可以按Ctrl+C停止监控

## 涨速计算原理
涨速 = (当前价格 - 上次价格) / 上次价格 / 时间间隔(分钟) * 100%

这个指标可以帮助识别短期内快速上涨的股票，捕捉市场热点。
